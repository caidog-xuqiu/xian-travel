# 当前版本亮点（可用于汇报/面试）

- 单 Agent 状态图已落地，不再是单纯接口串联 workflow。
- Agent V3 引入 Planning Loop V1，支持动作级决策闭环。
- LLM 参与“解析 + 选优 + 动作”，但不直接替代硬规划器。
- 候选方案层具备差异增强，提升“可选空间”的真实价值。
- 选优层支持后置复核，避免明显违背核心意图的方案被选中。
- selector 稳定性已增强，支持网络异常重试与本地约束回退。
- 动态搜索与策略矩阵联动，搜索方向可随用户意图变化。
- 支持澄清后继续执行，thread_id 可恢复中断会话。
- SQLite 持久化已接入，覆盖 checkpoint / memory / logs。
- 可读文案层可将结构化 itinerary 渲染为用户友好中文建议。
- 系统全链路具备回退机制，保证接口在外部服务异常时可用。
